Я напомню про **тернарный оператор** — присвоение переменной одного из двух значений в зависимости от условия:
```py
i = 5
a = i - 1 if i < 0 else i + 1
```
Что аналогично коду:
```py
i = 5
if i < 0:
  a = i - 1
else:
  a = i + 1
```
